구글 계정만 있으면 완전 무료로 무한 자동생성됩니다.
신용카드 등록 불필요 · 하루 2,500문제까지 무료 · 키는 이 기기에만 저장됩니다.
발급 방법 (2분)
1. aistudio.google.com/apikey 접속
2. 구글 계정 로그인
3. "Create API Key" 클릭
4. 키 복사 후 아래 입력 → 저장
💡 Claude.ai에서 사용할 때는 키 없이도 자동생성됩니다.
Netlify · PWA · 폰에서 직접 열 때만 키가 필요합니다.
📱 홈 화면에 앱 추가 (PWA)
이 앱을 스마트폰 홈 화면에 추가하면 일반 앱처럼 사용할 수 있습니다.
스플래시 화면, 전체 화면, 오프라인 기본 문제도 지원합니다.
설치 방법
🤖 안드로이드 (Chrome): 주소창 오른쪽 ⋮ 메뉴 → "홈 화면에 추가"
🍎 아이폰 (Safari): 하단 공유 버튼 □↑ → "홈 화면에 추가"
💻 PC (Chrome): 주소창 오른쪽 ⊕ 아이콘 → "설치"
🏪 Google Play Store 등록 방법 ▼
Netlify에 배포한 웹사이트를 TWA (Trusted Web Activity) 방식으로 Play Store에 등록할 수 있습니다. 코드 재작성 없이 가능합니다.
1
먼저 Netlify에 배포 → URL 확보 (홈 탭 참고)
2
play.google.com/console → 개발자 계정 등록 ($25 일회성)
3
pwabuilder.com 에서 Netlify URL 입력 → Android 패키지 자동 생성
4
생성된 AAB 파일을 Play Console에 업로드 → 검토 (보통 1~3일)
5
앱 심사 통과 후 스토어에 무료 등록 가능
⚠️ 단, Play Store 등록 앱에서 API 키가 소스에 노출됩니다. 공개 배포 시에는 별도 백엔드 서버(Netlify Functions 등) 로 API를 중계하는 것을 권장합니다.
